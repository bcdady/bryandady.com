## About This Resume Format

This resume is automatically rendered from structured JSON data, which provides several advantages:

- **Consistency**: Follows industry-standard JSON Resume Schema
- **Maintainability**: Single source of truth for resume data
- **Portability**: Can be easily converted to other formats (PDF, HTML, etc.)
- **Version Control**: Changes are tracked in git with full history
- **Automation**: Can be automatically updated and deployed

### Technical Implementation

The resume is rendered using a custom React component built for Docusaurus, with:
- Responsive design that works on desktop and mobile
- Print-friendly CSS for generating PDFs
- Semantic HTML structure for accessibility
- Professional styling with modern typography

### Data Source

The underlying resume data is stored in [`Bryan_Dady.resume.json`](./Bryan_Dady.resume.json) and follows the [JSON Resume Schema v1.0.0](https://raw.githubusercontent.com/jsonresume/resume-schema/v1.0.0/schema.json).
